Properties

Property Value Description
color color name or #rgb sets the foreground color
background color color name or #rgb sets the background color
font size px, em, rem the size of the font
font style normal, italic, oblique the style of the font
font weight normal, bold, lighter, bolder, numeric (100-900) the thickness of the font
text-transform none, uppercase, lowercase, capitalize, full-width overrides text values
text-decoration none, underline, overline, line-through for text decoration
font-family font 1st option, 2nd, 3rd selects the font of the text
text-shadow px + color sets a shadow behind the text
text-align left, right, center, justify text positioning
line-height px, em, rem, raw unit acts as multiplier sets height of lines
letter-spacing px, em, rem spaces between letters
word-spacing px, em, rem spaces between words

Examples

p{
color: red;
background-color: #ffffff;
font-size: 18px;
font-style: bold;
font-weight: lighter;
text-transform: capitalize;
text-decoration: underline;
font-family: Helvetica, Arial, sans-serif;
text-shadow: 2px blue;
text-align: center;
line-height: 1.2em;
letter-spacing: 1.4px;
word-spacing: 2em;
}
